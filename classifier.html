Summary

ℹ About Classification▸

Core Recoverable: Classic tokens frozen by new program. Unfreeze via token metadata delegate removal.

Generations: mpl-core assets in known staking pool. Unfreeze via freeze_delegate plugin removal.

Legacy (Dead): Frozen by deleted legacy program (bytecode = NULL on-chain). Cannot be recovered.

Other Pool: Generations in unknown pool. May be recoverable but requires manual verification.

Compressed: Compressed assets not currently supported for recovery.

Unclassified: Unknown status. Requires manual investigation.

Client-side classification only. No data transmitted. For recovery, see /recovery.html
